# o-nav [![Build Status](https://secure.travis-ci.org/Archetype-CSS/o-nav.png?branch=master)](http://travis-ci.org/Archetype-CSS/o-nav) [![Built with Grunt](https://cdn.gruntjs.com/builtwith.png)](http://gruntjs.com/)

Nav object to convert an an `ol` or `ul` into a horizontal, un-styled list for
building UI components.

## Installation
  * [Bower](http://bower.io): {{coming soon}}
  * Git: `git clone https://github.com/Archetype-CSS/o-nav.git`

## Use
  * `test-title--structure` - structure styles for the test title
  * `test-title--skin` - skin styles for the test title
  * `test-description--structure` - structure styles for the test description
  * `test-description--skin` - skin styles for the test description
  * `test--structure` - structure styles for the test
  * `test--skin` - skin styles for the test

## Run the Test Locally

```bash
git clone https://github.com/Archetype-CSS/o-nav.git
cd o-nav
npm install
grunt
```
